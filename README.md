# Barcode-generator
Generate code 128 barcode print page

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](https://raw.githubusercontent.com/s9801077/Barcode-generator/master/LICENSE)

## Usage

設定參數 --> 點擊預覽 --> 點擊列印

Notd: 列印時請將邊界設為零
